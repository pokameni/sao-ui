# ⚔ SAO UI — Minecraft Mod 🧩

A community-led initiative aiming to continue and modernize the original SAO-UI project for newer Minecraft versions and modloaders.

**At this time, no code, assets, ports and no development has started and no releases are available.**

The purpose of this repository is to serve as the foundation for a potential future continuation of the project.

## 👀 Vision

* Bring SAO-UI to modern Minecraft versions
* Support modern modding platforms and versions
* Preserve the spirit and appearance of the original project
* Ensure long-term maintainability
